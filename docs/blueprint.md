# **App Name**: eBikeNow

## Core Features:

- User Authentication: Secure user login and registration with email and password.
- Bike Availability: Display available e-bikes on a map with real-time status.
- Rental Management: Allow users to start and end rentals, tracking time and location.
- Payment Processing: Integrate secure payment gateway for rental fees.
- Rental Cost Estimation: The app provides an AI-powered tool that will use rental time, distance, and demand in order to accurately estimate the price of a ride
- Admin Dashboard: Provide admins with a dashboard to manage users, bikes, and rentals.
- Firestore Integration: Utilize Firestore to store and manage user, bike, and rental data.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to convey energy and environmental consciousness.
- Background color: Light gray (#E0E0E0) to provide a neutral and clean backdrop.
- Accent color: Teal (#008080) to highlight interactive elements and calls to action.
- Body font: 'PT Sans', a modern yet warm sans-serif font for readability.
- Headline font: 'Space Grotesk', a futuristic font to convey speed.
- Use minimalist, line-style icons to represent bike status and rental information.
- Clean, card-based design with clear information hierarchy.
- Smooth transitions and loading animations to enhance user experience.
